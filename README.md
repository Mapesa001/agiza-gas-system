# Agiza Gas System

Agiza Gas System is a comprehensive platform designed to streamline the gas supply chain. The system enables wholesalers to place orders directly with manufacturers and facilitates retailers to procure gas efficiently from wholesalers.

## Features
- **Order Management**: Place, track, and manage gas orders.
- **User Roles**: Separate portals for manufacturers, wholesalers, and retailers.
- **Real-Time Notifications**: Alerts for order status and stock updates.
- **Secure Transactions**: Ensure safe and seamless operations.
- **Responsive Design**: Accessible on desktops, tablets, and mobile devices.

## Technologies Used
- **Frontend**: React.js
- **Backend**: Node.js (JavaScript)
- **Database**: MySQL
- **Package Manager**: Yarn
